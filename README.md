# Chrono Task - Smart Calendar & Todo Manager
Transform your productivity with ChronoTask! A visually stunning fusion of calendar and task management that helps you organize your life day by day.

## Why Chrono Task?
- Visual Planning: See your tasks directly on the calendar
- Smart Organization: Group tasks by dates effortlessly  
- Instant Updates: Real-time synchronization between calendar and tasks
- Beautiful Interface: Enjoy working with a clean, modern design

## Perfect For
- Students managing assignments
- Professionals tracking deadlines
- Anyone wanting to organize daily activities
- Productivity enthusiasts

## Tech Stack
- Pure HTML, CSS, JavaScript
- Modern CSS Grid & Flexbox
- ES6+ JavaScript Features
- Responsive Web Design
