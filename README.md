# Chrono Task - Smart Calendar & Todo Manager
Transform your productivity with ChronoTask! A visually stunning fusion of calendar and task management that helps you organize your life day by day.

## Why Chrono Task?
- Visual Planning: See your tasks directly on the calendar
- Smart Organization: Group tasks by dates effortlessly  
- Instant Updates: Real-time synchronization between calendar and tasks
- Beautiful Interface: Enjoy working with a clean, modern design

## Perfect For
- Students managing assignments
- Professionals tracking deadlines
- Anyone wanting to organize daily activities
- Productivity enthusiasts

## Chrono Task Features
- Add a new task and select the date when the task will be completed.
- Tasks added on a specific date can be accessed by pressing that date on the calendar, and the "My To-Do List" box will display all tasks added on that date.
- The "All Tasks" box will show all tasks that have been added. To view all of them, press the "Show All" button and scroll down to see all the tasks that have been added.
- If a user adds a task on a specific date, a red dot will appear below the date, indicating that the user has a task to complete on that date.
- If a task is completed, the user can press the light purple button or the checkmark icon next to the task text to mark the task as finished.
- Users can delete an added task by pressing the trash can icon next to the checkmark icon.
- New feature: Users can switch the theme from dark to light and vice versa by pressing the button in the lower right corner.
